## Instruction
1. 安装django, pymysql, mysqlclient...
   ```shell
    pip3 install Django pymysql mysqlclient
   ```
2. ```shell
    cd (GitHub repo 地址)/2020-DBLAB/demo
   ```
3. ```shell
    python3 manage.py migrate
   ```
4. ```shell
    python3 manage.py runserver
   ```
5. 浏览器访问`http://127.0.0.1:8000/`
